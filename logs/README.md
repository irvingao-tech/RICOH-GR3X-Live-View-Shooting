# Logs Directory

本目录用于保存实机问题调查、串口日志摘要和复现步骤。不要把大体积原始日志、Wi-Fi 密码或完整设备私密标识直接提交。

## 建议文件命名

```text
YYYY-MM-DD-short-topic.md
```

## 建议内容

- 问题现象
- 复现步骤
- 相关日志
- 相关代码位置
- 初步结论
- 未确认事项
- 后续验证步骤

## 脱敏规则

- Wi-Fi 密码必须删除。
- BLE MAC/BSSID 可保留末 2-3 字节或按问题需要保留，但需避免公开隐私设备信息。
- 大文件放外部，不直接进 Git。

## 后续 Codex 修改代码时必须注意

- 先看 logs 中是否已有同类问题。
- 修复后补充“验证结果”和“是否回归”。
